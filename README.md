# Task List Project

A sleek and intuitive web application designed to help users manage their daily tasks efficiently. Users can view their current tasks, easily add new ones, and remove completed tasks with a single click.

## Features

* **View Tasks:** A clean, organised dashboard displaying all current tasks.
* **Add Tasks:** Quickly insert new items into your to-do list using the dedicated input and add button.
* **Remove Tasks:** Clean up your list instantly by removing tasks once they are completed.
